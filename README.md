## 电子科技大学教务系统美化插件

> 让老旧的教务系统焕发青春。  
> 项目官网：[blog.simplenaive.cn/uestc_helper]()

### 开发指南

```bash
# 1. 安装开发依赖
npm install

# 2. 打开Chrome，以开发着模式加载 dev/devExtension　文件夹

# 3. 开始开发
npm start

```

### 文档指南

项目的`docs`[文件夹](https://github.com/Yidadaa/UESTC_Helper/tree/master/docs)下有关于项目的若干文档，敬请查阅。
